# TicTacToeLuxoft
For Luxoft, Odessa, Ukraine (2019)

<br />

## Quick start
###For Linux:
1. For build in Terminal type: `make`
2. For start `./bin/TicTacToeLuxoft`
3. *For clean `make clean`

<br />

###For Windows:
Just use Visual Studio by opening `TicTacToeLuxoft.sln`
